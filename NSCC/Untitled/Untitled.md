War Dialer is a script that tells the model to dial a range of phone numbers defined by the user and identifies those numbers that connect to remote computers. 

An early form of an automated scanner. 

In the early 1980s, majority of servers ran on UNIX platforms 
SYstem Administrators created shell scripts that let them check their networks. 


As internet grew, more devices became connected. 

Their checks automated the processes of dinding network weaknesses. 

The time of all this was the time for the appearance of the Worm. 


Ping scanning : Demonstrates whether a remote host is active by sending ICMP echo request packets to that host. 


Some hosts will actively ignore ping requests. 

## UDP Scanning

Examines the status of UDP ports on a target system
The scanner sends a 0-byte UDP packet to all the ports on the target


## IP Protocol Scanning -sO

Examines a target for supported IP protocols (TCP, ICMP<IGMP, etc.)

Scanner transmits IP packets to each protocol on the target

## TCPDump :


Command line packet analyzer. It can read packets from a network interface card.
