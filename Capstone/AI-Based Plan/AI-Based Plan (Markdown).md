# Capstone Project Plan: Intrusion Detection System

## Milestones and Tasks

### Milestone 1: Project Planning and Initial Setup
>[!note] **Due Date**: Jan 24, 2025

- **Task 2.3**: Generate Report Framework

>[!info] **Due Date**: Mar 24, 2025

### Milestone 5: User Interface and Final Report
>[!note] **Due Date**: Apr 4, 2025

- **Task 5.1**: Develop User Interface
  - Create a simple user interface for running assessments
- **Task 5.2**: Finalize Report Generation
  - Ensure the final report includes all necessary information

### Milestone 6: Advanced Features and Optimization

## Weekly Scrum Meetings

### Scrum Report Structure

For each meeting, include:

1. **Task at Hand**: Describe the current task each team member is working on.
2. **Progress**: Outline the progress made on the task.
3. **Challenges Encountered**: List any challenges faced.
4. **Resolution**: Describe how challenges were resolved or mention if they are unresolved.
5. **Future Plan**: Provide a plan for the upcoming week.

> - **Future Plan**: Continue developing scripts for other tools and integrating with BASH scripts.
