

# Goal Statement


#### Developing a **Goal Statement** : 


1. Ask yourself "Is it a true desire?"
2. Write the goal down! it becomes substantial when it is writtem
3. Identify the obstacles you will face, and what will help you overcome them. 
4. Make a plan!




#### Goal Statement Requirements :


1. State the goal
2. make sure it is a **specific** goal!
3. Make sure it is **measurable**!
4. Make sure it is **realistic**!
5. Make sure there is a **timeline** and a **deadline**!



Example :

"We will have a man on the moon and safely returned to earth in this decade!"
Kennedy


#### Five Ws Question

1. Who is involved?
2. What do I want to acheive?
3. Where is the goal to be achieved? 
4. When should it be achieved?
5. Why do I want to achieve this goal? 


#### Measurable 


you need a criteria for making progress. 



#### Achievable


Do we have the means to achieve the goal?
Has others successfully accomplished this goal? 


#### Realistic 

Is this goal within reach?
Can it be done by April ? 
Is there a milestone that can be had in April?
Are you able to commit to achieve this goal? 


#### Timely

You want to have some time bounds on the goal.
Does the goal have a deadline? you can have a few for each stage of the project.
